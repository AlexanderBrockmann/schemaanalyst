NEGATIVE TESTS
(reset tables each time)

* PK--UNIQUE tests
	useful tests plus additionally adds a not null for unique col

* NULL tests
	adds a null for the col concerned

* FK tests
	useful tests plus additionally adds a not null for unique col

* UNIQUE col tests
    unique / matches (unique and non-unique), additionally adds a not null for unique col

* EXPR tests
	negates expression.

For each new tests, try all of the old ones to see if they satisfy predicate already.
